# Tetris 游戏

一个经典的俄罗斯方块游戏，使用 Godot 4.6 引擎开发。

## 游戏介绍

这是一个经典俄罗斯方块游戏的现代化实现，使用 Godot 4.6 引擎开发。游戏具有流畅的动画、多种控制方式、分数系统和保存功能。

## 玩法说明

### 游戏目标
- 控制下落的方块，将它们排列成完整的水平线
- 完成的水平线会被消除，获得分数
- 防止方块堆积到屏幕顶部
- 尽可能获得更高的分数

### 控制方式

**键盘控制：**
- **左箭头 / A**：向左移动
- **右箭头 / D**：向右移动
- **下箭头 / S**：向下移动（软降）
- **上箭头 / W / E**：顺时针旋转
- **Q**：逆时针旋转
- **Esc**：暂停游戏

**手柄控制：**
- **方向键左/右**：向左/右移动
- **方向键下**：向下移动
- **A 按钮**：顺时针旋转
- **B 按钮**：逆时针旋转
- **Start 按钮**：暂停游戏

### 游戏规则
1. 方块从屏幕顶部下落
2. 使用控制键移动和旋转方块
3. 完成水平线可以消除并得分
4. 随着等级提升，游戏速度会加快
5. 方块堆积到屏幕顶部时游戏结束
6. 旋转被阻止时会导致方块低于底部边界
7. 按住下箭头可以快速下落

### 游戏特色
- 经典俄罗斯方块玩法
- 键盘和手柄支持
- 分数系统和等级进阶
- 保存/加载功能
- 暂停菜单
- 游戏结束画面
- 最高分记录
- 放弃运行确认对话框
- 改进的碰撞检测
- 流畅的旋转机制

## 技术栈

- **游戏引擎**：Godot 4.6
- **开发语言**：GDScript
- **图形**：2D 图形
- **部署**：WebAssembly (WASM)

## 本地运行

1. 下载并安装 Godot 4.6
2. 克隆仓库：
   ```bash
   git clone https://github.com/zml9167/tetris.git
   ```
3. 在 Godot 中打开项目（选择 `project.godot` 文件）
4. 点击播放按钮开始游戏

## 保存和加载

- 游戏会在暂停时自动保存进度
- 可以从标题画面继续游戏
- 可以从标题画面放弃当前运行（需要确认）
